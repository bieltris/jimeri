# Checklist Beta 0.9

## Fluxos criticos

- Login com credenciais validas
- Logout e retorno para a tela de login
- Cadastro de cliente
- Edicao de cliente
- Cadastro de produto
- Cadastro de categoria durante a edicao de produto
- Registro de pagamento
- Cancelamento de pagamento
- Lancamento de pedido
- Cobranca por WhatsApp

## Mobile

- Abrir a tela de login no celular
- Testar instalacao do app pela tela de login
- Abrir o app instalado pela tela inicial
- Validar barras do navegador no Safari e no Chrome mobile
- Conferir o carrinho flutuante no registro de pedidos
- Conferir modais com teclado aberto

## Conferencias de dados

- Divida do cliente sobe ao lancar pedido
- Divida do cliente desce ao registrar pagamento
- Cliente ativo aparece primeiro nas listagens
- Produtos fora da venda nao aparecem no lancamento de pedido
- Categoria do produto aparece corretamente nas listagens

## Falhas que nao podem passar

- Clique duplo gerando pedido duplicado
- Clique duplo gerando pagamento duplicado
- Tela branca apos erro de rede
- Sessao expirada sem redirecionamento
- Modal escondendo campos importantes no mobile
